# Arabic-Language-for-minecraft-bedrock
بكل اختصار هو مشروع هدفه اضافة اللغة العربية الى ماينكرافت

كيف يمكنك المساهمة في تطويرها ؟

اذا اردت ان تساهم في التطوير ادخل الى موقع https://github.com/3li-1/Arabic-Language-for-minecraft-bedrock/ ومن ثم ادخل الى ملف الذي مساره هكذا /text/ar_SA.lang وعدله وقبل التعديل اذا بغيت تضيف اي ترجمة لازم اول تحوله الى عربي يدعم ماينكرافت عبر موقع https://www.arabic-text.com/ ومن ثم اضافته الى ملف /text/ar_SA.lang يعني مثلا كذا 

0- example=example

 1- لازم تترجم معنى الترجمه مثلا example يكون معناها "مثال"
 
 2- تروح لموقع arabic-text.com وتحولها من "مثال" الى "لﺎﺜﻣ" ومن ثم تضيفها الى الترجمة يعني مثلا example=لﺎﺜﻣ ولازم الي ما قبل اليساوي يكون نفس اسمه فقط الي بعد اليساوي هو المترجم 

3- كيف الان احطها في المشروع الرسمي ترجمتي ؟
